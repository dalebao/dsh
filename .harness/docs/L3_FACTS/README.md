# L3 Facts — Stable Operational Knowledge

Each file covers one domain of stable, slow-changing operational truth.

| File | Content |
|---|---|
| `build_test.md` | Build commands, test runner, CI job definitions |
| `deploy.md` | Deployment workflow, environments, rollback procedures |
| `config.md` | Configuration files, env vars, secrets handling |

**Rule:** Only update L3 facts when the underlying system changes. These are the most stable layer.
